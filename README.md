# SQL Commercial Sales Performance Analysis

This project uses Microsoft SQL Server to analyse a large synthetic UK commercial sales database.

The database contains customers, products, sales representatives, monthly targets, orders, order items, returns, marketing campaigns and campaign responses.

The project demonstrates:

- Relational database design
- SQL joins and aggregations
- Common table expressions
- Window functions
- Sales and customer analysis
- Product and regional performance
- Profitability and discount analysis
- Returns analysis
- Marketing campaign analysis
- Data-quality validation
- Power BI-ready views

> This project uses purpose-built synthetic data. It does not contain confidential customer or company information.

## Project Status
